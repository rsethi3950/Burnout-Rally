# Burnout-Rally
## Save the Boat

- configure openGL in C++/VsCode
- Create project
- Go to Project Options-> Parameters -> Linker
- Add following to linker:

        -lfreeglut

        -lopengl32

        -lglu32

        -lwinmm
- Compile and Run
